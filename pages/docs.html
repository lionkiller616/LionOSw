infoIntroduction to LionOS

Welcome to the official documentation for LionOS. This guide aims to provide you with all the information you need to install, use, and enjoy our operating system.

LionOS is a modern, Linux-based operating system designed with a focus on performance, simplicity, and a beautiful user experience. Inspired by the elegance of macOS and the robustness of established Linux distributions, LionOS offers a unique blend of power and ease-of-use.

Our Philosophy

We believe that your operating system should be an intuitive tool that empowers you, rather than a complex system that gets in your way. Our core principles include:

  • Minimalism: A clean, clutter-free interface that promotes focus.
  • Performance: Optimized for speed and efficiency on a wide range of hardware.
  • Elegance: Thoughtful design in every pixel, from icons to animations.
  • Security: Robust security features and privacy-respecting defaults.
  • Community: Built by and for a passionate global community of users and developers.

Whether you are new to Linux or a seasoned veteran, we hope you find LionOS to be a refreshing and productive environment.

memorySystem Requirements

To ensure a smooth experience with LionOS, your system should meet the following minimum requirements. Recommended specifications are for optimal performance.

Minimum Requirements:

  • Processor: 1 GHz dual-core processor (x86-64)
  • RAM: 2 GB
  • Storage: 20 GB of free hard drive space
  • Graphics: Graphics card capable of 1024x768 resolution
  • Media: USB port or DVD drive for installation media

Recommended Specifications:

  • Processor: 2 GHz quad-core processor (x86-64) or better
  • RAM: 4 GB or more
  • Storage: 40 GB SSD (Solid State Drive) for faster performance
  • Graphics: Modern graphics card with good Linux driver support (NVIDIA, AMD, Intel)
  • Internet: Broadband internet connection for updates and online features

Note: LionOS is designed to be lightweight, but performance will vary based on your specific hardware configuration.

build_circleInstallation Guide

This section provides a step-by-step guide to installing LionOS on your computer. Please back up any important data before proceeding.

  1. Download the ISO: Get the latest LionOS ISO image from our Downloads page. Verify the checksum to ensure integrity.
  2. Create Bootable Media:
    • USB Drive: Use a tool like BalenaEtcher, Rufus (Windows), or `dd` (Linux/macOS) to write the ISO image to a USB drive (at least 4GB recommended).
    • DVD: Burn the ISO image to a blank DVD.
  3. Boot from Media: Restart your computer and enter your BIOS/UEFI settings (usually by pressing DEL, F2, F10, F12, or ESC during startup). Configure your computer to boot from the USB drive or DVD.
  4. Start the Installer: Once LionOS boots from the media, you'll see a welcome screen. Choose the "Install LionOS" option.
  5. Follow On-Screen Instructions: The installer will guide you through:
    • Language selection
    • Keyboard layout
    • Timezone
    • Partitioning: You can choose to erase the entire disk and install LionOS, or manually partition if you plan to dual-boot. Be careful with partitioning as it can lead to data loss if done incorrectly.
    • User account creation (username, password).
  6. Installation Process: The installer will copy files and configure the system. This may take some time.
  7. Restart: Once the installation is complete, remove the installation media and restart your computer.

Congratulations! You should now be able to boot into your new LionOS installation.

rocketFirst Boot & Setup

After successfully installing and restarting, you'll be greeted by the LionOS desktop.

Initial Setup (if applicable):

  • You may be prompted to connect to a Wi-Fi network.
  • A welcome application might guide you through some initial configurations or highlight key features.
  • It's highly recommended to check for and install any available system updates soon after your first boot. You can usually do this through a "Software Updater" application or via the terminal.

Explore the desktop, familiarize yourself with the application launcher, system tray, and settings panel. Welcome to LionOS!

desktop_windowsDesktop Overview

LionOS features the "Elysium" desktop environment, designed for beauty and efficiency. Key components include:

  • The Dock: Quick access to your favorite and running applications.
  • Top Panel: Contains the application menu (for focused apps), system status indicators (network, sound, battery), notifications, and a clock/calendar.
  • Application Launcher: Access all installed applications. Usually opened via an icon on the dock or a keyboard shortcut.
  • File Manager: "Files" provides an intuitive way to browse and manage your documents and folders.
  • System Settings: Centralized control panel for configuring all aspects of your OS.

Spend some time exploring to see how these elements work together to create a seamless workflow.

buildTroubleshooting

If you encounter issues, here are some initial steps:

  • Consult the LionOS Community: Check our forums or chat groups. Someone may have already solved a similar problem.
  • Check System Logs: Use tools like `journalctl` (if systemd-based) to view system messages for errors.
  • Hardware Compatibility: Ensure your hardware is well-supported by the Linux kernel version LionOS uses.
  • Search Online: General Linux troubleshooting resources can often be helpful.

For specific issues, providing detailed information (error messages, steps to reproduce, hardware details) will help the community assist you.
